# Trailer Display Test Tool

Windows app for testing TrailerDisplay LED boards via TCP.

## Protocol
```json
{"trailer_id":"001","trailer_number":"М530СМ","trailer_info":["4101","2005"]}
{"trailer_id":"001","trailer_number":"М530СМ","trailer_info":""}
```

## Ports
- **5001** — Unload (В)
- **5002** — Load (П)
